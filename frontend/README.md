# 🎵 Music App

A full-stack Music Application built using **React (Frontend)** and **Python (Backend)** that allows users to play and manage music with a modern UI.

---

## 🚀 Features

* 🎶 Play songs
* ⏸ Pause / Resume music
* ⏭ Next / Previous track
* 📃 Playlist support
* 🎨 Modern responsive UI
* ⚡ Fast frontend with React
* 🔗 Backend API integration

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/kishore-SY/music-app.git
cd music-app
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

### Backend Setup

```bash
cd backend
python main.py
```
